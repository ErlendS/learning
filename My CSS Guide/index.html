The Box Model

What is it?

The CSS Box Model describes how CSS structure elements created in a document tree. Each element is placed in it's own rectangular box, and each box contains a content area, and have the option to have a padding, border and/or margin area.

The classic box model, the content area is the only necessary of the four, the rest allows for more style options. Note that the margin area is always transparent.

Incase you need a reminder of what an element is, it comes from HTML and usally consist of a start tag "<" and and end tag"/>" paragraphs <p>, headers <h1-h6> and lists <ul>, <li> are all HTML elements. Everything of content on your website are created my elements. And the box model describes how they are styled and placed on the site.

But enough theory, let's look at an example of the box model in action! This example will show nested elements in an unordered list.

Here is the code snippet for this example 

  .ul {
    background-color: yellow;
    margin: 0;
    padding: 10px;
  }
  .li_1 {
    background-color: hotPink;
    margin:10px;
  }
  li_2 {
    margin: 0;
  }
  • I'm my own element with my own box model, I'm nested inside the Ul element. That means I'm a child element of the Ul
  • I'm also a child element, my width is the same as the one above me, but my margins are set to zero.
